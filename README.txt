To run the project follow the instructions given below:

1.Create a table 'addbook' with the following structure in ORACLE

	fieldname	type
	
	name		varchar2(20)
	mob		varchar2(15)
	email		varchar2(50)

2.Open the project in Netbeans IDE.

3.Resolve the jar file issues if any.(ojdbc14.jar)

4.Change the connection string values(url,username,password) according to your database configuration in \MVCAddressBook\src\java\com\myapp\struts\db\DbConnect.java file
